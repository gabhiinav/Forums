# Forums - Where Your Interests Meet

Tech Stack:

- Next.js
- TailwindCSS
- NextAuth w/ Google OAuth
- Prisma
- PostgreSQL
- Redis
- Uploadthing 
- Vercel

### Setup .env.local

```js
DATABASE_URL=
NEXTAUTH_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=
REDIS_URL=
REDIS_SECRET=
```

### Start

```shell
npm run dev
```
